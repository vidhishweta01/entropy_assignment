## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby: 3.0.0
Rails: 7.0.3

### Setup

Install gems with:

```
bundle install --path vendor/bundle
```

Setup database with:

```
   rails db:create
   rails db:migrate
```

### Usage

Start server with:

```
    rails s
```

Open `http://localhost:3000/` in your browser.
